FinAnalyzer - Step by Step Run Guide

Use this file first. It explains what to install, which file to run, and which file to close.

Before You Run

Install Prerequisites

  1. Install Node.js LTS (includes npm): https://nodejs.org
  2. Keep this full project folder on the computer.
  3. If importing directly from Tally:
    • Open Tally Prime
    • Enable XML connectivity (default port 9000)
Internet is required the first time for npm dependency install.
Start

Which File to Run (Desktop Recommended)

  1. Double-click: run_tauri.bat
  2. Desktop window will open directly (no browser needed).
  3. In the app, choose:
    • Import from Tally, or
    • Import Tally Source File
Browser fallback mode still available:
run_software.bat then open http://127.0.0.1:5173
If desktop config is not available, run_tauri.bat auto-falls back to browser mode.
Stop

Which File to Close

  1. Tauri desktop mode: double-click close_tauri.bat
  2. Browser mode: double-click close_software.bat
Always close using the matching close file before shutting down or restarting.
Share Data

How to Share Between Users

  1. Inside app, click Export Tally Source File
  2. Send generated .tsf file to other user
  3. Other user imports with Import Tally Source File

TSF Raw to Excel (New)

  1. Open app first page
  2. Use Sub Feature: TSF Raw to Excel
  3. Select shared .tsf file to download raw .xlsx
Package

Create Desktop Installer

  1. Preferred: double-click package_tauri.bat
  2. Wait for build to complete.
  3. Share installer from src-tauri\target\release\bundle\nsis folder.

Main run file: run_tauri.bat | Main close file: close_tauri.bat
